Manufacturing Production & Scrap Analysis Dashboard

This interactive dashboard built using Microsoft Power BI analyzes manufacturing operations across multiple departments to monitor production efficiency, machine utilization, and scrap generation.

The dashboard provides a comprehensive overview of key manufacturing metrics including Issue Quantity, Production Quantity, Scrap Quantity, Department-wise Consumption Rate, and Machine Hours. KPI cards at the top summarize overall operational performance, enabling quick evaluation of production health.

Several visualizations help identify patterns and inefficiencies:

Department-wise Machine Consumption Rate to compare operational usage across Turning, Quality, Packaging, Milling, and Assembly departments.
Scrap Quantity Analysis to identify which departments contribute most to material waste.
Machine Consumption vs Scrap Quantity Comparison to evaluate the relationship between machine usage and scrap generation.
Component-wise Scrap Tracking to monitor scrap contributions by different components.
Issue Quantity Distribution showing department-level material usage.

Interactive filters such as Component ID slicers allow users to dynamically explore the data and perform deeper analysis.

This dashboard helps manufacturing managers and analysts:

Monitor production efficiency
Identify high-scrap departments
Track machine utilization
Improve operational decision-making

Overall, the project demonstrates how Power BI can transform raw manufacturing data into actionable insights through interactive visual analytics.
